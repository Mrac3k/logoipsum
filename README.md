# Logoipsum Landing Page

A responsive landing page for a subscription business platform.

## Tech Stack

- HTML5
- CSS3 (Flexbox, Grid, Custom Properties)
- No frameworks, no JavaScript

## Features

- Fully responsive design (mobile, tablet, desktop)
- Fluid typography with clamp()
- Adaptive header with menu wrap
- Smooth scroll behavior

## Breakpoints

- 1024px — tablet
- 767px — mobile

## Getting Started

Clone the repo and open `index.html` in your browser.

```bash
git clone https://github.com/Mrac3k/logoipsum.git
```
